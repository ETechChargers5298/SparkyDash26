# SparkyDash26
Dashboard to visualize ETech Chargers scouting data for the 2026 FRC season

Notes:
* Get `.db` file from Scoutradioz and name it `scouting_2026.db`
* Use `streamlit run app.py` to deploy local code.  Make sure `streamlit` and other packages installed
* Code deplyed as webapp on Streamlit at `insert link here`
* For more detailed setup, [click here](https://docs.google.com/document/d/1ngwI4-w987vJ05lRaehP5DCMXdv80h2WXWsXazjMwQY/edit?tab=t.0#heading=h.95iat3jlg0zi).
